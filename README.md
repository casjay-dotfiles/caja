## caja  
  
File Manager  
  
Automatic install/update:  
  
```
bash -c "$(curl -LSs https://github.com/dfmgr/caja/raw/master/install.sh)"
```
  
Manual install:  
requires:  
```apt install caja```  
```yum install caja```  
```pacman -S caja```  
  
```
mv -fv "$HOME/.config/caja" "$HOME/.config/caja.bak"
git clone https://github.com/dfmgr/caja "$HOME/.config/caja"
```
